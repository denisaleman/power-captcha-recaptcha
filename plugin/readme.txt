== Description ==

Protect your WordPress/WooCommerce website from spam, brute-force attacks, and fake accounts with Google reCAPTCHA.

Power Captcha reCAPTCHA supports 3 Google reCAPTCHA types integrated into 6 common WordPress forms, including login and comment forms, and 7 WooCommerce forms.

== 3 captcha types ==

- **Score based (v3) captcha.** A seamless detection.
- **"I'm not a robot" captcha checkbox.** Verification requests with a challenge.
- **Invisible reCAPTCHA**. Improved, challenge-based captcha without a checkbox.

== 6 WordPress forms ==
- **Login form.**
- **Register form.**
- **Comment form.**
- **Lost password form.**
- **Reset password form.**
- **Register form.**

== 7 WooCommerce forms ==
- **Login form.**
- **Register form.**
- **Checkout form.**
- **Review form.**
- **Reset password form.**
- **Lost password form.**

== Installation ==
 
1. Upload the `power-captcha-recaptcha` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3  Sign up in [Google reCAPTCHA console](https://www.google.com/recaptcha/admin) to get Site Key and Secret Key for your site. (it's FREE)
4. Enter your Site Key and Secret Key in Settings -> Power Captcha menu in WordPress.
5. Enable the desired captchas.

== Frequently Asked Questions ==

= Is plugin relying on 3rd party as service? =

Yes. The functionality of the plugin relies on the integration of a third-party service, specifically Google reCAPTCHA, into the WordPress website. For a thorough understanding of the service, including its terms of service and privacy policy, please refer to the official documentation provided at the following link: https://www.google.com/recaptcha/about/.
